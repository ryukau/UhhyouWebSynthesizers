Uhhyou Web Synths Collection

Some of the recent generators don't work on Firefox 104. Currently recommending to use Chrome or Safari (Blink or WebKit based browser). See this link for more details.

Impulse Response

These are intended to be used with external IR convolver.

FDNReverb
Reverb using feedback delay network.
Pulseverb
Reverb using modulated band-limited impulse train (BLIT).
Freeverb
Reverb using Freeverb algorithm.
